[DABE] Migrate virtual machines from on-premises environments to Google Cloud

Managed by Google Cloud Partners
7 activities
Last updated 7 days
Start
Designed for partner delivery practitioners in the DABE program, this learning path provides the baseline delivery skills required to execute physical and virtual machine migrations from on-premises environments to Google Cloud. Practitioners will master critical discovery and assessment workflows using the Migration Center, while developing actionable skills to establish essential hybrid and multicloud networking topologies, high-throughput VPNs, and internal load balancing. Completion ensures practitioners can configure core compute, networking, and persistent storage elements to deliver reliable, structured lift-and-shift migrations for clients.
